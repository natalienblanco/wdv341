WDV341 Intro PHP

  • Version Control Software: A system that assists with managing the changes to your PHP code.

  • Add: The "git add" command stages the changes to commit.

  • Commit: The "git commit" command saves changes to the local repository. This becomes a part of the project's history.

  • Push: The "git push" command is used to upload content in your local repository to a remote repository, so that others can access them.

  • Pull: The "git pull" command fetches & merges changes from a remote repository to a local one. This updates your local codebase with the most recent changes made by others.

  • Clone: The "git clone" command is used to make a copy of a remote repository in your local machine.
